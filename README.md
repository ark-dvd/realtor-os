# Merrav Berko Real Estate

## Setup

1. `npm install`
2. Create `.env.local`:
```
NEXT_PUBLIC_SANITY_PROJECT_ID=6hdpy5q3
NEXT_PUBLIC_SANITY_DATASET=production
SANITY_API_TOKEN=your_token_here
```
3. `npm run dev`
